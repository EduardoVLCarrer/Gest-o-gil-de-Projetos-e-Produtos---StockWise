STOCKWISE
Plataforma de Análise de Ações
=========================================

Descrição do Projeto

O StockWise é uma plataforma desenvolvida como MVP para auxiliar investidores na pesquisa e análise de ações do mercado financeiro. O objetivo é reunir, em uma interface simples e intuitiva, informações relevantes sobre empresas listadas em bolsa, permitindo que o usuário pesquise ativos e visualize indicadores que auxiliem sua tomada de decisão.

Nesta etapa do projeto foram desenvolvidos os artefatos de planejamento, incluindo a Lean Inception, definição do MVP, Product Backlog, Sprint 1 e os wireframes da interface do usuário.
